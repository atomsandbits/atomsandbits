## Software stack

atoms+bits is a Node.js application that runs on the following software:

* Linux/macOS
* Meteor 1.6.1.1+
* Node.js 9.11.1+
* Git 2.8.4+
* MongoDB 3.6.3+
* Elasticsearch 6.2.4+
* TensorMol 0.2
* Psi4 1.1
* PySCF 1.4.7

Architecture information is available [here](doc/development/architecture.md).
