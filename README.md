## Requirements

~~~sh
sudo pip install pycodestyle
sudo pip install pydocstyle
sudo pip install pylint
sudo pip install yapf
sudo pip install tensorflow
~~~

- meteor
- node.js and npm
- mongodb
- TensorMol

## Setup

Set custom environment variables:
~~~sh
touch variables.env
~~~

Install dependencies:
~~~sh
npm install
~~~

## Development

Run all services:
~~~sh
npm start
~~~
